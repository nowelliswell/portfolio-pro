### Step 1: Submit ke Google Search Console (5 menit)

1. **Buka Google Search Console**
   - URL: https://search.google.com/search-console
   - Login dengan Google account

2. **Add Property**
   - Klik "Add Property"
   - Pilih "URL prefix"
   - Masukkan: `https://noel-porto.vercel.app/`
   - Klik "Continue"

3. **Verify Ownership** (Pilih salah satu metode)
   
   **Metode A - HTML Tag (Paling Mudah):**
   - Copy verification meta tag dari Google
   - Buka file `index.html`
   - Replace `YOUR_VERIFICATION_CODE_HERE` dengan code dari Google
   - Commit & push ke GitHub
   - Tunggu Vercel redeploy (1-2 menit)
   - Klik "Verify" di Google Search Console
   
   **Metode B - HTML File:**
   - Download verification file dari Google
   - Upload ke folder `public/`
   - Commit & push
   - Klik "Verify"

4. **Submit Sitemap**
   - Setelah verified, klik "Sitemaps" di sidebar kiri
   - Masukkan: `sitemap.xml`
   - Klik "Submit"
   - Status akan berubah jadi "Success" dalam beberapa jam

5. **Request Indexing**
   - Klik "URL Inspection" di top bar
   - Masukkan: `https://noel-porto.vercel.app/`
   - Klik "Request Indexing"
   - Tunggu konfirmasi
